# Child Health Agent Prototype

An interactive static prototype for a crowd-sensing, trustworthy inquiry agent that supports risk communication for child and adolescent growth, development, and eye health.

## What it demonstrates

- Growth and puberty information collection
- Eye health and myopia-prevention behavior fields
- Inquiry-first Agent interaction
- Risk communication summary
- Personalized science communication suggestions
- Follow-up and feedback loop

This prototype is for health education and risk communication only. It does not provide diagnosis or treatment.
